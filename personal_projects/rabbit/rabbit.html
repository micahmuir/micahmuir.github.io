Rabbit Robot V2

Building on the lessons learned from the first iteration, this second version of the rabbit robot focuses on aesthetics and functionality. The goal was to create something that actually looks like a rabbit while incorporating innovative features like environmental scanning through functional "ears."

Goals

  • Expand on the lessons learned with the first iteration
  • Streamline software pipelines
  • More focus on end aesthetics. Can we make this thing actually look like a rabbit?

Ideation

The key question was: can we make this thing look like a rabbit? What would the ears do? The breakthrough came with the idea of making the ears actually useful for scanning the environment. This provided the perfect excuse to experiment with pogo pins and magnetic connectors.

Execution

The design process involved careful consideration of both form and function. The challenge was fitting all the necessary components from V1 into a more aesthetically pleasing rabbit-like form factor.
